Android Native Client

Install the dedicated Android App for optimized scanner performance, system auto-start, and persistent local caching bounds.

Download APK Installer
  1. Once the file downloads, tap nexova-pos-release.apk to launch the package manager.
  2. If prompted, enable "Install from Unknown Sources" in your device security settings.
  3. Complete installation, launch Nexova POS, and enter a friendly name to pair the device.

Windows Desktop Client

Install the native Windows client for multiple display support (KDS, CFD), scale integrations, and high-performance offline access.

Download MSI Installer
  1. Download the installer package.
  2. Run the NexovaPOS-1.0.0.msi and follow setup steps.
  3. Start Nexova POS from the desktop shortcut to pair your register.

Progressive Web App (PWA)

Run the zero-trust POS directly in your browser with full offline support and instant homescreen launching.

Launch PWA Client
  1. For iOS Safari: Tap the Share button, scroll down, and select "Add to Home Screen".
  2. For Android Chrome: Tap the three dots menu, and select "Install App".
  3. Launch the app from your homescreen to enable secure, standalone full-screen cashier grids.
